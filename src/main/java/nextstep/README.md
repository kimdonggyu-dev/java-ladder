## 요구사항

- [ ] 입력
    - [ ] 참가자 리스트 입력
    - [ ] 실행 결과 입력
    - [ ] 최대 사다리 높이 입력
    - [ ] 결과를 보고 싶은 사람 입력
        - [ ] 사람
        - [ ] all
- [ ] 출력
    - [ ] 사람 이름 출력
    - [ ] 사다리 게임 결과 출력
    - [ ] 실행 결과 출력
    - [ ] 결과 보고 싶은 사람 출력
- [ ] 로직
    - [ ] 사람 이름 최대 5글자 제한
    - [ ] 사다리 생성
        - [ ] 라인이 겹치지 않게 구현
    - [ ] 사람 이름에 따른 사다리 폭 조절
    - [ ] 특정 사다리의 결과를 출력한다.
